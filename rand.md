# Random Useful Commands

## Bulk Change Passwords
```
chpasswd
# Then, enter passwords one line at a time using format <user>:<password>
# You can also pass a file
passwords > chpasswd
```

## Check Services Using Ports
```
sudo netstat -plant
```
